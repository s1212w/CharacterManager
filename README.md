# CharacterManager

## 주제
1. 캐릭터 관리 프로그램


## 요구사항
1. 캐릭터의 이름, 직업, 능력치 정보를 입력받는다.
2. 여러 캐릭터를 한번에 관리할 수 있다.
3. 캐릭터의 능력치가 단순 숫자가 아닌 아이콘으로 표시되어 한눈에 비교할 수 있다.
4. 장비를 추가하여 캐릭터의 능력치를 상승시킬 수 있다.
